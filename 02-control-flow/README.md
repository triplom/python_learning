## Key Concepts

- `if condition:` — executes block if condition is `True`
- `elif` — chained condition (else if)
- `else` — fallback when all conditions are False
- `for x in sequence:` — iterate over any iterable
- `range(start, stop, step)` — generate a numeric sequence
- `while condition:` — repeat until condition is False
- `break` — exit loop immediately
- `continue` — skip to next iteration
