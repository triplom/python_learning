## Key Concepts

| Structure | Ordered | Mutable | Duplicates | Syntax |
|-----------|---------|---------|------------|--------|
| list | Yes | Yes | Yes | `[1, 2, 3]` |
| tuple | Yes | No | Yes | `(1, 2, 3)` |
| dict | Yes (3.7+) | Yes | Keys: No | `{"a": 1}` |
| set | No | Yes | No | `{1, 2, 3}` |
